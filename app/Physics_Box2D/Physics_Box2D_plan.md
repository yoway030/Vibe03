# Physics_Box2D 모듈 개발 계획

## 모듈 개요
Box2D.NET 물리 엔진을 래핑하여 게임에서 쉽게 사용할 수 있는 물리 시뮬레이션 모듈

## 개발 목표
- Box2D.NET 라이브러리를 사용한 2D 물리 시뮬레이션 제공
- 간단하고 직관적인 API 제공
- 멀티 온라인 땅따먹기 게임에 필요한 물리 기능 지원

## 주요 기능

### 1단계: 기본 물리 시뮬레이션 (완료)
- [x] PhysicsWorld 클래스 구현
- [x] 정적 바디(Static Body) 생성 - 벽, 지면 등
- [x] 동적 바디(Dynamic Body) 생성 - 플레이어, 오브젝트
  - [x] 박스 형태
  - [x] 원형 형태
- [x] 물리 시뮬레이션 스텝 실행
- [x] 위치 및 회전 정보 조회
- [x] 속도 설정 및 조회

### 2단계: 힘과 충격
- [x] 힘(Force) 적용
- [x] 충격(Impulse) 적용
- [x] 밀도, 마찰력, 반발력 설정

### 3단계: 바디 관리
- [x] 바디 생성/삭제
- [x] 바디 존재 여부 확인
- [x] 모든 바디 목록 조회

### 4단계: 고급 기능 (향후 계획)
- [ ] 충돌 감지 및 콜백
- [ ] 조인트(Joint) 시스템
- [ ] 커스텀 Shape 지원
- [ ] 레이캐스팅(Raycasting)
- [ ] 영역 쿼리(Area Query)
